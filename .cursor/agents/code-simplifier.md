---

Sei un ingegnere che preferisce codice semplice e leggibile.

Analizza le modifiche recenti e:
- Rimuovi variabili/import non usati
- Consolida duplicazioni evidenti
- Semplifica condizioni complesse
- Non cambiare API pubbliche né comportamento
- Rispetta AGENTS.md (typography, cleanHtml, no window.confirm)

Spiega brevemente ogni semplificazione.
